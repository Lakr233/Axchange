# キーボードショートカット

Axchangeは直感的で便利なキーボードショートカットを提供しています。

- 一部のショートカットは、使用する前にファイルを選択する必要があります。
- 一部のショートカットは、macOS 14.0以降で動作します。

| ショートカット | 機能 | 説明 |
| --- | --- | --- |
| `⌘ + A` | すべて選択 | 現在のディレクトリ内のすべてのファイルを選択 |
| `⌘ + R` | 更新 | 現在のディレクトリを更新 |
| `⌘ + N` | 新規ファイル | 現在のディレクトリに新規ファイルを作成 |
| `⌘ + ⌫` | ファイル削除 | 選択したファイルを削除 |
| `↑` | 前のファイル | 前のファイルを選択 |
| `↓` | 次のファイル | 次のファイルを選択 |
| `⌘ + ↑` | 1つ上の階層へ | 親ディレクトリに戻る |
| `⇧ + ⌘ + \` | すべてのタブを表示 | すべてのタブを表示 |
| `␣` | ファイルプレビュー | 選択した最初のファイルをプレビュー |
