# バージョン更新

## 1.6

- 日本語翻訳を追加しました。

## 1.5

**2024年12月1日**

- App Store以外でダウンロード可能なバージョンを追加しました。
- デバイスペアリングガイドページを追加しました。

## 1.3

**2024年11月15日**

- ドラッグ＆ドロップのサポートを追加しました（`macOS 14.0`以上が必要）。

## 1.2

**2024年11月12日**

- macOS 15より前のバージョンでサービスがクラッシュする問題を修正しました。
- 一括転送が正しくキャンセルできない問題を修正しました。
- クイックルック機能を追加しました（`macOS 14.0`以上が必要）。
- より多くの操作のための右クリックメニューを追加しました。
- ナビゲーションとファイル開くためのダブルクリック簡易操作機能を追加しました。
