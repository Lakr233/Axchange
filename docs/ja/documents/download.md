# ダウンロード

ダウンロードを開始するには、ファイルを選択してメニューバーの`ダウンロード`ボタンをクリックするか、ファイルをFinderにドラッグします。

![ダウンロード](../../res/SCR-20241212-nmnj.jpeg)

ダウンロード処理中、インターフェースが切り替わりダウンロード進捗状況が表示されます。複数のファイルが選択されている場合、ファイルは順番にダウンロードされ、全体の進捗状況が表示されます。

![ダウンロード進捗](../../res/SCR-20241212-nmmo.jpeg)

## 注意事項

- ダウンロードは既存のファイルを上書きし、既存のフォルダとマージします。
- 複数選択されたファイルの場合、ダウンロード順序を指定することはできません。
