# 快捷键

Axchange 提供了丰富且直观的快捷键。

- 部分快捷键需要选中文件以后才可使用。
- 部分快捷键需要 macOS 14.0 以上版本才可使用。

| 快捷键 | 功能 | 说明 |
| --- | --- | --- |
| `⌘ + A` | 全选 | 选中当前目录下的所有文件 |
| `⌘ + R` | 刷新 | 刷新当前目录 |
| `⌘ + N` | 新建文件 | 在当前目录下新建一个文件 |
| `⌘ + ⌫` | 删除文件 | 删除选中的文件 |
| `↑` | 上一个文件 | 选中上一个文件 |   
| `↓` | 下一个文件 | 选中下一个文件 |
| `⌘ + ↑` | 返回上级目录 | 返回上级目录 |
| `⇧ + ⌘ + \` | 显示所有标签页 | 显示所有标签页 |
| `␣` | 预览文件 | 预览选中的第一个文件 |
