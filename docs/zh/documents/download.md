# 下载

单机列表中的内容来选中目标文件。按住 `⌘` 或者 `⇧` 来选中多个内容。在选中需要下载的文件后，工具栏中的`下载`按钮将会被启用。

![选中文件](../../res/SCR-20241212-nmnj.jpeg)

点击该按钮可将文件保存到指定位置。

![下载](../../res/SCR-20241212-nmmo.jpeg)

对于多个选中的项目，下载过程中会有进度提示。

## 注意事项

- 下载操作会覆盖已存在的文件，合并已有文件夹。
- 下载文件会使用 ADB 默认的文件权限。若要保留文件权限，建议使用压缩文件格式。
- 对于多个选中的文件，下载顺序无法指定。
