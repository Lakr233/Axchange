# Keyboard Shortcuts

Axchange provides a range of intuitive and convenient keyboard shortcuts.

- Some shortcuts require selecting a file before they can be used.
- Some shortcuts require macOS 14.0 or later to work.

| Shortcut | Function | Description |
| --- | --- | --- |
| `⌘ + A` | Select All | Select all files in the current directory |
| `⌘ + R` | Refresh | Refresh the current directory |
| `⌘ + N` | New File | Create a new file in the current directory |
| `⌘ + ⌫` | Delete File | Delete the selected file |
| `↑` | Previous File | Select the previous file |
| `↓` | Next File | Select the next file |
| `⌘ + ↑` | Go Up One Level | Return to the parent directory |
| `⇧ + ⌘ + \` | Show All Tabs | Display all tabs |
| `␣` | Preview File | Preview the selected first file |
